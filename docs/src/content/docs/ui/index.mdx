Vizb generates a single self-contained HTML file with a Vue.js-powered interactive UI. No server needed — open the file in any browser.

## Chart Types

Vizb supports three chart types, all rendered in a single view:

<CardGrid>
  <Card title="Bar Chart" icon="bars">
    Default chart type. Compare values across categories. Supports linear and logarithmic scale.
  </Card>
  <Card title="Line Chart" icon="analytics">
    Track trends across X-axis values. Best for sequential data like input sizes or concurrency levels.
  </Card>
  <Card title="Pie Chart" icon="puzzle">
    Show proportional distribution. Useful for comparing relative sizes of benchmark results.
  </Card>
</CardGrid>

Control which charts appear with `--charts`:

```bash
vizb bench.txt -c bar,line -o output.html
```

When your data has a z-axis, bar and line charts render in **3D** instead of 2D.
See [3D Charts](/ui/3d-charts).

## Scale Options

Toggle between linear and logarithmic Y-axis scale:

```bash
# Linear (default)
vizb bench.txt -o output.html

# Logarithmic — better for high-variance data
vizb bench.txt --scale log -o output.html
```

<Aside type="tip">
  Use `--scale log` when benchmark values span several orders of magnitude (e.g., 1 to 1,000,000).
</Aside>

## Tooltips & Axis Sums

Hovering a data point shows its value alongside **axis-sum** totals. 2D charts
add per-series totals after each series name and a labeled x-marginal at the
bottom of the tooltip. 3D charts add `Σ z` / `Σ x` / `Σ y` marginals — see
[3D Charts](/ui/3d-charts). Tooltips follow light/dark mode automatically.

## Fullscreen

Every chart has a **fullscreen** button in its toolbar (next to export). Click it
to expand the chart to the full viewport, and again to exit.

## Single-File Output

The generated HTML is fully self-contained:
- Vue.js app embedded inline
- Chart data embedded as JSON
- No external dependencies or network requests
- Works offline

Open with any browser — no build step, no server.
