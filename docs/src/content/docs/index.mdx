## Features

<CardGrid stagger>
  <Card title="Multi-Chart" icon="analytics">
    Bar, line, and pie charts in a single view. Sort, swap axes, and toggle logarithmic scale.
  </Card>
  <Card title="Merge & Compare" icon="code-branch">
    Merge benchmarks from multiple runs, machines, or releases. Tag with version labels for cross-release comparison.
  </Card>
  <Card title="Smart Grouping" icon="list-format">
    Extract dimensions from benchmark names using pattern matching or regex. 1D, 2D, or 3D data — vizb handles it.
  </Card>
  <Card title="CI Ready" icon="seti:puppet">
    First-class GitHub Action. Run benchmarks, tag releases, merge history, and deploy — all from your workflow.
  </Card>
  <Card title="Interactive UI" icon="laptop">
    Vue.js-powered interface with sortable charts, axis swapping, scale toggles, and JPEG export. All in a single HTML file.
  </Card>
  <Card title="Flexible Input" icon="document">
    Reads benchmark output from Go, Rust (Criterion, Divan), and JavaScript (Vitest, Tinybench). Auto-detects text and JSON formats. Pipe from CLI or read from files.
  </Card>
</CardGrid>
