Sample inputs live under [`examples/`](https://github.com/goptics/vizb/tree/main/examples). CI converts each file to JSON, merges them into one HTML page per category, and deploys to [vizb.goptics.org/examples/](https://vizb.goptics.org/examples/). Open a specific dataset with `?id=<id>` — each matrix entry sets a stable id via the action `id` input (forwarded as `--id`).

## Adding an example

Same steps for every category — see [examples/README.MD](https://github.com/goptics/vizb/blob/main/examples/README.MD):

1. Add the file under `examples/{category}/`
2. Add a matrix entry in `.github/workflows/deploy-examples-{category}.yml`
3. Push to `main` — CI rebuilds that dashboard
