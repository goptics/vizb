## Density notes

| File | Points | Use when |
|------|--------|----------|
| `noise-surface.csv` | 441 (21×21) | Fast height-field demo |
| `noise-grid.csv` | 9,261 (21³) | Manageable voxel + visualMap |
| `noise-grid-41.csv` | 68,921 (41³) | Full ECharts-scale grid |

Prefer the 21³ grid for everyday loads; use 41³ when you need every voxel.
