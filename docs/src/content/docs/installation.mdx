## Quick Install

<QuickInstall />

<Aside type="note">
  The installer downloads the latest binary from GitHub Releases to `~/.local/bin` on Linux/macOS or `%LOCALAPPDATA%\vizb` on Windows. No admin required.
</Aside>

## Docker

Pull the image and start the API on port 8080:

```bash
docker pull goptics/vizb
docker run --rm -p 8080:8080 goptics/vizb
```

The image starts `vizb serve` bound to `0.0.0.0` inside the container. You can
also start it with the repository's Compose configuration:

```bash
docker compose up -d
```

For CLI use, mount the current directory so Vizb can read inputs and persist
outputs:

```bash
docker run --rm -v "$PWD:/data" -w /data goptics/vizb bar sales.csv -o out.html
```

Pipe input with `-i`; the mounted directory still keeps the generated file:

```bash
cat sales.csv | docker run --rm -i -v "$PWD:/data" -w /data goptics/vizb bar -o out.html
```

<Aside type="caution">
  The API has no built-in authentication. Do not expose it publicly without your own authentication, TLS, and access controls. See [`vizb serve`](/commands/serve/) for details and port remapping.
</Aside>

## Go Install

```bash
go install github.com/goptics/vizb@latest
```

<Aside type="note">
  Building from source requires **Go 1.26+**. Install Go from the [official site](https://go.dev/dl) or via [webi](https://webinstall.dev/go/). Pre-built binaries and the install script need no Go runtime.
</Aside>

## Download Binary

Pre-built binaries for Linux, macOS, and Windows (amd64 and arm64) are available on the [releases page](https://github.com/goptics/vizb/releases).

<Tabs syncKey="os">
  <TabItem label="Linux" icon="linux">
  ```bash
  # Download (adjust version)
  curl -sfL https://github.com/goptics/vizb/releases/download/v0.10.3/vizb@0.10.3-linux-amd64.tar.gz | tar xz
  chmod +x vizb
  sudo mv vizb /usr/local/bin/
  ```
  </TabItem>
  <TabItem label="macOS" icon="apple">
  ```bash
  # Apple Silicon (M1/M2/M3)
  curl -sfL https://github.com/goptics/vizb/releases/download/v0.10.3/vizb@0.10.3-darwin-arm64.tar.gz | tar xz
  chmod +x vizb
  sudo mv vizb /usr/local/bin/

  # Intel
  curl -sfL https://github.com/goptics/vizb/releases/download/v0.10.3/vizb@0.10.3-darwin-amd64.tar.gz | tar xz
  chmod +x vizb
  sudo mv vizb /usr/local/bin/
  ```
  </TabItem>
  <TabItem label="Windows" icon="seti:windows">
  1. Download `vizb@0.10.3-windows-amd64.zip` from [releases](https://github.com/goptics/vizb/releases)
  2. Extract `vizb.exe`
  3. Add to a directory in your `PATH`
  </TabItem>
</Tabs>
