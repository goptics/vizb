Installation

Quick Install

Section titled “Quick Install”
  • Linux / macOS
  • Windows

Note

The installer downloads the latest binary from GitHub Releases to ~/.local/bin on Linux/macOS or %LOCALAPPDATA%\vizb on Windows. No admin required.

Go Install

Section titled “Go Install”

Note

Building from source requires Go 1.24+. Install Go from the official site or via webi. Pre-built binaries and the install script need no Go runtime.

Download Binary

Section titled “Download Binary”

Pre-built binaries for Linux, macOS, and Windows (amd64 and arm64) are available on the releases page.

  • Linux
  • macOS
  • Windows
  1. Download vizb@0.10.3-windows-amd64.zip from releases
  2. Extract vizb.exe
  3. Add to a directory in your PATH
